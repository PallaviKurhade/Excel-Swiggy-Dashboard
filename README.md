# Excel-Swiggy-Dashboard

Swiggy Sales Analysis Dashboard

📌 Project Overview

This Excel dashboard analyzes Swiggy sales performance across different states, food categories, restaurants, and time periods. The dashboard helps track sales trends, customer ratings, order volumes, and business performance through interactive visualizations.

🎯 Business Problem

Food delivery businesses need to understand:

- Which states generate the highest sales?
- Which food categories perform best?
- How do sales change over time?
- What are customer rating trends?
- Which restaurants contribute most to revenue?

This dashboard provides answers through data-driven analysis.

🔍 Key Insights

- Non-Veg food generated the highest sales contribution.
- Karnataka achieved the highest total sales.
- Weekend sales outperformed weekday sales.
- Customer ratings remained consistently high.
- Sales trends showed stable business growth.
